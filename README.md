# Wyxion

Универсальное GUI-приложение для оптимизации сетевого трафика и обхода систем глубокого анализа пакетов (DPI) на Windows. Программа объединяет в себе гибкий клиент управления службами обхода ограничений и стабильный локальный MTProto-прокси для Telegram.

> [!WARNING]
> ### АНТИВИРУСЫ
> 
> WinDivert может вызвать ответ антивируса. WinDivert — это инструмент для перехвата и фильтрации трафика, выбранный для блокировки работы. Замена `iptables` и `NFQUEUE` в Linux, которых нет под Windows. Он может использоваться как хорошими, так и плохими программами, но сам по себе не является вирусом. Драйвер `WinDivert64.sys` подписан для возможности загрузки в 64-битное ядро Windows.
> 
> \* *Выдержка из `readme.md` репозитория [bol-van/zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle)*
> 
> В случае возникновения проблем с антивирусом добавьте утилиту в исключения, либо отключите обнаружение PUA (потенциально нежелательных приложений). Например, в Kaspersky есть галочка «Обнаруживать легальные программы, которые злоумышленники часто используют для распространения вреда». Рекомендуется аккуратно настроить параметры исключений, но если вы не до конца поняли, что делаете, рекомендуется оставить обнаружение PUA включенным.

## Особенности

* **Высокая скорость** — минимальные задержки при обработке и маршрутизации сетевых пакетов.
* **Локальный шлюз** — все данные обрабатываются непосредственно на вашем устройстве без передачи третьим лицам.
* **Стабильное подключение** — надежная обфускация трафика и поддержка интеграции с Cloudflare Proxy / Worker.
* **Интуитивный интерфейс** — быстрое переключение пресетов, встроенная диагностика сети и удобный менеджер Telegram Fix.

## Использование

1. Перейдите в раздел релизов и скачайте последнюю версию Wyxion.
2. Запустите приложение от имени Администратора (требуется для регистрации системной службы и запуска драйвера WinDivert).
3. Выберите оптимальный пресет обхода из списка и нажмите кнопку **Установить службу**.
