# Wyxion

Программа для операционной системы Windows, которая помогает восстановить стабильный доступ к заблокированным ресурсам, сайтам и голосовым каналам (таким как YouTube, Discord и другие) путем обхода систем глубокого анализа пакетов (DPI) вашего интернет-провайдера. Дополнительно в программу встроен инструмент Telegram Fix – персональный локальный прокси-сервер, нормализующий работу мессенджера Telegram в условиях жестких сетевых ограничений.

---

## Как это устроено

*   **Локальная работа** – все операции по изменению сетевых пакетов происходят на вашем компьютере. Программа не является VPN-сервисом, не передает ваш трафик на сторонние сервера, не собирает личные данные и не снижает скорость вашего интернет-соединения.
*   **Высокая скорость** – пакеты данных модифицируются «на лету» непосредственно процессором вашего ПК, что исключает появление задержек при просмотре видео или в онлайн-играх.
*   **Стабильное подключение** – надежная обфускация (маскировка) исходящих сетевых запросов под обычный безопасный HTTPS-трафик, а также встроенная интеграция с Cloudflare Proxy / Worker для бесперебойной работы.
*   **Удобное управление** – запуск и настройка всех функций осуществляются в один клик: быстрое переключение готовых стратегий обхода, встроенная сетевая диагностика и автоматическая интеграция прокси в Telegram.

> [!WARNING]
> ### ПОЧЕМУ АНТИВИРУС МОЖЕТ ПРЕДУПРЕЖДАТЬ ОБ ОПАСНОСТИ?
> Для низкоуровневого перехвата и изменения сетевых пакетов Wyxion использует легитимный драйвер WinDivert (системный файл WinDivert64.sys имеет официальную цифровую подпись).
> 
> Поскольку этот драйвер умеет перенаправлять сетевой трафик внутри системы для его последующей модификации, защитные программы (особенно Windows Defender и Kaspersky) часто классифицируют его как программу повышенного риска (детект вроде Not-a-virus:RiskTool.Multi.WinDivert или HackTool).
> 
> Это ложное срабатывание эвристического анализатора: антивирус видит техническую возможность перехвата запросов и блокирует утилиту в целях профилактики. Программа не содержит в себе скрытого вредоносного кода.
> 
> **Что делать в случае блокировки:**
> 1. Добавьте исполняемый файл Wyxion.exe и папку, в которой он находится, в список исключений (белый список) вашего антивируса.
> 2. Если антивирус все равно блокирует запуск драйвера, временно отключите в его настройках обнаружение потенциально нежелательных программ (PUA / Riskware / Легальный софт, который может использоваться злоумышленниками).

## Использование

Для запуска и настройки программы выполните следующие шаги:

### 1. Скачивание приложения
Перейдите в раздел **Releases** на этой странице GitHub и скачайте актуальную версию файла `Wyxion.exe`.

### 2. Размещение
Поместите скачанный файл в любую удобную папку на вашем компьютере. Не рекомендуется запускать программу прямо из папки «Загрузки» или системных папок вроде *Program Files*, так как программе требуется беспрепятственный доступ для записи логов и локальных конфигураций.

### 3. Запуск от имени Администратора
Нажмите на файл `Wyxion.exe` правой кнопкой мыши и выберите пункт **«Запуск от имени администратора»**. Это обязательное условие, так как программе необходимы привилегии для установки системного драйвера WinDivert и регистрации службы Windows.

### 4. Выбор профиля
В левой панели приложения выберите один из готовых пресетов обхода (для большинства провайдеров оптимально подходит стандартный пресет `general`). Если у вашего провайдера этот профиль не работает, остановите службу и попробуйте выбрать альтернативные пресеты из списка.

### 5. Активация обхода
Нажмите кнопку **«Установить службу»**. В консоли отобразится лог установки, а в панели мониторинга статус службы изменится на зеленый маркер **«Запущена»**.

### 6. Проверка работы
Откройте браузер и проверьте доступность сайтов. Если вам также требуется настроить обход для мессенджера, перейдите во вкладку **Telegram Fix**, нажмите кнопку **«Начать»** и затем **«Открыть в Telegram»** для автоматического применения локального прокси.
